# TGAT Project

TGAT is a Python desktop application for controlled traffic generation, basic network performance analysis, and attack simulation in a lab environment. The project includes a PyQt6 GUI, packet generation with Scapy, ping-based analysis, and supporting documentation for academic review.

## Files

- `main.py` - main PyQt6 GUI application
- `traffic_generator.py` - traffic generation logic
- `attack_simulator.py` - SYN flood and DDoS simulation helpers
- `performance_analyzer.py` - latency, jitter, loss, and throughput analysis
- `TGAT_Documentation.docx` - project documentation
- `Evaluation Report.pdf` - evaluation report
- `CONTRIBUTION.REPORT.PDF` - contribution report

## Requirements

Install dependencies with:

```bash
pip install -r requirements.txt
```

## Notes

Use this project only in an authorized lab or educational environment.
